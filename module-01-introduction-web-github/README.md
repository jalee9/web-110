# Module 1: Introduction to Web Development & GitHub

## Solution Review: Hello Web

This folder contains the reference solution for the Module 1 Hello Web lab.

In this lab, students practiced the first steps in the web development workflow:

- Completing beginner HTML/CSS practice
- Creating a simple `index.html` file
- Testing the file in a browser
- Creating a GitHub repository
- Uploading project files to GitHub

## What to Review

As you review the solution, pay attention to:

- The basic HTML document structure
- The use of the `head` and `body` sections
- The page title
- The main heading
- Paragraph elements
- File naming
- Repository organization

## Key Reminder

This first lab is intentionally simple. The goal is to practice creating a valid HTML file and saving it in GitHub. Styling and publishing will be introduced later in the course.

## Why This Lab Matters

This lab was not just about adding text to a webpage.

It reinforced foundational web development concepts:

### 1. HTML Structure

You practiced organizing content using headings, paragraphs, and lists.

Good webpages are organized and readable. Structure helps users understand content more easily.

### 2. Semantic Organization

You learned that HTML is not just about appearance.

HTML helps describe the purpose of content:

- Headings organize information
- Paragraphs explain ideas
- Lists group related information

Later, CSS will control appearance.

### 3. Readable Code

You practiced indentation and formatting to make your code easier to read.

Good code is not only code that works—it is code that other people can understand.

### 4. Comments

You practiced using comments to label major sections of your webpage.

Comments help developers stay organized and understand code later.

### 5. Version Control Foundations

You continued practicing beginner-friendly GitHub workflows by updating your repository and documenting changes.

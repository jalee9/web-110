# WEB 110: Web Development Fundamentals

This repository contains instructor-created solution examples and learning reviews for WEB 110: Web Development Fundamentals.

The purpose of this repository is to provide students with additional reinforcement after completing course labs and activities. These examples are designed to help students review key concepts, compare their work to a reference solution, and better understand the web development workflow.

## How to Use This Repository

Students should use these files as learning support, not as a replacement for completing their own work.

For each module, review the solution after you have attempted the assignment or after the assignment has closed in Blackboard. Pay attention to the structure of the files, the comments in the code, and the learning review notes.

Each module may include:

- A reference solution
- Annotated code examples
- A learning review
- Notes about common errors or misunderstandings
- Suggestions for continued practice

## Course Focus

WEB 110 introduces foundational skills in:

- HTML structure
- CSS styling
- File and folder organization
- GitHub repositories
- GitHub Pages publishing
- Accessibility basics
- Responsive design
- Ethical use of AI and coding support tools

## Important Reminder

These examples are provided to support learning. Students are still responsible for writing, testing, validating, and explaining their own code.

Do not copy and submit solution files as your own work. Instead, use them to strengthen your understanding and identify areas for improvement.
